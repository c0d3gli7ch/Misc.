## `SWIFT: SOCIETY FOR WORLDWIDE INTERBANK FINANCIAL TELECOMMUNICATION.`

<br>

> [!NOTE]
> SWIFT Code: 8-11 Characters.
> +  First 4: Bank or Institude Code
> +  Later 2: Country Code
> +  Last 2: Location/City Code.
